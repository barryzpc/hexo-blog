如何隐藏HTTP响应头里的server和X-Powered-By

2019-11-21

场景描述

很多时候我们为了安全考虑，需要隐藏我们环境的一些版本号，如：server、X-Powered-By

未隐藏时http响应头中包含了如下信息

apache 隐藏 server

修改httpd.conf 设置 

ServerSignature Off
ServerTokens Prod

nginx 隐藏 server

修改nginx.conf 在http里面设置 

server_tokens off;

隐藏X-Powered-By

修改 php.ini 文件 设置 

expose_php = Off
